# Sevvi — Know Before You Mix

## Run in VS Code
1. Open `sevvi2` folder in VS Code
2. Install **Live Server** extension
3. Right-click `index.html` → Open with Live Server
4. Opens at http://127.0.0.1:5500

## Pages
- **Home** — Landing page with features
- **Checker** — Drug interaction checker
- **History** — Past checks (saved locally)
- **About** — App info

## Features
- Tamil + English toggle
- Voice input (Chrome)
- Read aloud results
- AI-powered for unknown combinations
- History saved in browser
